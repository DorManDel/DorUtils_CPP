# DorUtils_CPP

DorUtils_CPP is a personal C++ utility and learning library.

The goal of this project is not only to build reusable code, but also to deeply understand how data structures and algorithms work internally.

This project combines:

- C++ implementations
- written documentation
- runnable demos
- tests
- debug helpers
- interactive HTML visualizations

---

# Main Idea

Each topic in DorUtils is built as a complete learning unit.

A full module may include:

- implementation file (`.hpp`)
- explanation file (`.md`)
- runnable example (`.cpp`)
- test file (`.cpp`)
- visualization (`HTML/CSS/JS`)
- optional debug integration

This makes the project both a utility library and a personal study system.
